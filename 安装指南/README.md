## 推荐安装流程

1. 首先阅读 [快速安装指南](./快速安装指南.md) 了解基本步骤
2. 如需离线安装或迁移，请运行 [下载依赖包.bat](./下载依赖包.bat) 下载所有依赖
3. 详细问题请参考 [完整安装与迁移指南](./完整安装与迁移指南.md)
4. 如遇GPU识别问题，请参考 [GPU识别问题解决方案](./GPU识别问题解决方案.md)

## 系统要求

- 操作系统：Windows 10/11
- Python版本：3.10.x
- 硬盘空间：约2GB（包括依赖库）
- 可选GPU加速：NVIDIA显卡 + CUDA 11.8
